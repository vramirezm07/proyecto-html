Diseño y Desarrollo Web

Este repositorio está diseñado para complementar lo que se ha visto en clase, proporcionando recursos adicionales y ejemplos prácticos para ayudar a los estudiantes a comprender mejor los conceptos clave.

Guía de Navegación

Para facilitar tu estudio, hemos organizado el contenido en varias secciones. Puedes navegar a través de los siguientes enlaces para acceder a cada tema:

  • HTML: Aprende sobre la estructura básica de las páginas web y cómo utilizar etiquetas HTML para crear contenido.
  • CSS: Descubre cómo estilizar tus páginas web utilizando CSS para mejorar la apariencia visual y la usabilidad.
  • Cheat Sheet: Accede a una hoja de referencia rápida con atajos de teclado y comandos útiles para el desarrollo web.

Recursos Adicionales

Además de los temas principales, también hemos incluido recursos adicionales para profundizar en tu conocimiento:

  • Documentación de HTML en MDN
  • Documentación de CSS en MDN
  • Normalize.css: Una alternativa moderna y lista para HTML5 a los resets de CSS.
